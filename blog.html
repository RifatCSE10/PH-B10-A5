What is Document Object Model (DOM)

Document Object Model (DOM) is programming interface that we use to interact with HTML documet using JavaScript.

How do you select an element from the DOM?

Here are some common methods that is use for select DOM:
getElementById(id): Selects an element by its unique ID.
getElementsByClassName(className): Selects elements by their class name.
getElementsByTagName(tagName): Selects elements by their tag name.
querySelector(selector): Selects the first element that matches a CSS selector.
querySelectorAll(selector): Selects all elements that match a CSS selector.

What is event delegation in the context of the DOM , and why is it useful?

Event delegation refers to the process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated. It allows us to attach a single event listener for elements that exist now or in the future
